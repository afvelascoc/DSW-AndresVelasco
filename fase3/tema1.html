Frameworks CSS


Un Framework es un conjunto de herramientas, bibliotecas, convenciones y buenas prácticas diseñadas para encapsular tareas repetitivas en módulos comunes que son fáciles de reutilizar.

Teniendo en cuenta lo anterior, el Framework CSS es un conjunto de herramientas, hojas de estilo y buenas prácticas que permiten a los diseñadores web olvidarse de las tareas repetitivas y centrarse en los elementos únicos que pueden agregar valor a cada diseño.

Las hojas de estilo CSS son difíciles de organizar, mantener y reutilizar. Incluso los cambios de estilo menores requieren que escriba nuevas reglas CSS, que en algún momento pueden convertir su código en un lío complicado.

Las clases listas para usar son los componentes principales de todos los Frameworks CSS. Le permiten aplicar reglas de estilo predefinidas a elementos HTML, como márgenes, colores de fondo y otros.
Algunos Frameworks incluyen componentes prediseñados como menús, tarjetas o tablas. El uso de estos componentes le permite crear interfaces fáciles de usar sin mucho trabajo de su lado.


Algunas utilidades básicas del marco son:

  • Neutraliza el estilo predeterminado aplicado por el navegador. Esta es una hoja de estilo de reset.css común utilizada por todos los diseñadores profesionales.
  • El texto se procesa correctamente para que todo el contenido tenga exactamente el mismo aspecto en todos los navegadores, y se pueda adaptar para mejorar su accesibilidad y permitir su acceso en cualquier medio y / o dispositivo.
  • Cree cualquier estructura o diseño complejo de forma sencilla y asegúrese de que funcione en cualquier versión de cualquier navegador.


El Framework mas popular:

  • Bootstrap: Desarrollado por Twitter en 2010 combina CSS y JavaScript para diseñar elementos de página HTML. Es más que simplemente cambiar el color de los botones y enlaces. Esta es una herramienta que brinda interactividad en la página, por lo que proporciona una serie de componentes que facilitan la comunicación con los usuarios, como menús de navegación, controles de página, barras de progreso, etc.
